# Brushes

Brush strokes can be added in Graphic using the _**Brush Tool**_ from the _**Toolbar**_. You can create basic calligraphic brushes or pressure sensitive brushes. 

(image: brushes1.jpg  width: 750)

Brush attributes can be changed from the _**Tool Options bar**_ or using the (link: docs/brushes/customizing-brushes#brush-pane text: _**Brush Pane**_) from the Appearance panel.
