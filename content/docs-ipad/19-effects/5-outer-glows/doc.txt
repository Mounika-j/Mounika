# Outer Glows

The _**Outer Glow**_ effect is used to add an outer glow effect and to adjust its color and blur. Changes are applied to the currently selected objects.

Tap any effect to get access to its properties.  As with the other attributes, multiple _**Outer Glow**_ effects can be added to an object.

(image: outerglows1-ipad.jpg width:750)

1. **Color** - Set the color of the effect and its transparency.

2. **Edit** - Tap this button to duplicate, remove, edit or change the blending mode of the effect.

3. **Blur** - Set the amount of blur that should be added for this effect. A higher blur value will result in a larger and blurrier glow.
