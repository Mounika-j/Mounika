# Labels

Dimension labels are useful for creating designs such as floor plans, diagrams, or other technical illustrations. Use labels to measure the area of a room or label a series of objects.

(image: labels1-ipad.jpg width:750)

Using the (link: docs-ipad/labels/adding-labels text: _**Label pane**_) from the Appearance panel, you can apply a label on any type of shape or path. Labels can display dimension values or simple text.

(video: labels2-ipad.mp4 width:750)

Once applied, labels can be stylized using the (link: docs-ipad/labels/stylizing-labels text: _**Text Style pane**_) inside the Label panel.
