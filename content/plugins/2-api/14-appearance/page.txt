Title: Appearance

----

Text: 

An appearance object. This object manages the appearance items applied to a _(link: plugins/api/shape text: **Shape**)_.
&nbsp;
_Items_
- (link: plugins/api/appearance#items text: **items**)
- (link: plugins/api/appearance#addItem text: **addItem:**)

<a name="addItem"></a>
## addItem:
### _returns (link: plugins/api/appearance#AppearanceItem text: **AppearanceItem**)_; _input **String**_

Creates and returns a new appearance item object of the specified type, adding it to the appearance item stack.

<a name="items"></a>
## items
### _returns **Array**_

Gets the array of appearance items in the appearance object.

<a name="AppearanceStrokeDashArray"></a>
## dashArray
### _returns **Array**_; _settable_

Gets / Sets the stroke's line dash style. The array of _**Float**_ values adjusts the dash and gap spacing. Set this property to nil, or to an empty array, to remove the dash style.

<a name="AppearanceStrokeStrokeType"></a>
## strokeType
### _returns **String**_; _settable_

Gets / Sets the type of stroke - _**"center"**_, _**"inside"**_, or _**"outside"**_.

<a name="AppearanceStrokeLineCap"></a>
## lineCap
### _returns **String**_; _settable_

Gets / Sets the stroke's line cap style - _**"butt"**_, _**"round"**_, or _**"square"**_.

<a name="AppearanceStrokeLineJoin"></a>
## lineJoin
### _returns **String**_; _settable_

Gets / Sets the stroke's line join style - _**"miter"**_, _**"round"**_, or _**"bevel"**_.


<a name="AppearanceStrokeType"></a>

Gets the type of this appearance item - _**"stroke"**_
