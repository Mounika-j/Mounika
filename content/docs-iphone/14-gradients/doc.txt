# Gradients

Gradients can be used to stylize the _**Fill**_ of any object, even text. There are three types of gradients to choose from: _**Linear**_, _**Radial**_ and _**Angle**_ gradients. 

Use the (link: docs-iphone/gradients/customizing-gradients text: _**Gradient panel**_) to customize any type of gradient. Alternatively, you can select a pre-made gradient from the (link: docs-iphone/gradients/gradient-palette text: _**Gradient palette**_). 

(image: gradients1-iphone.jpg width: 750)

Additionally, you can adjust the position of your gradient using the (link: docs-iphone/gradients/gradient-tool text: _**Gradient tool**_).
