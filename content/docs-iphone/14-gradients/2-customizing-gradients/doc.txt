# Customizing Gradients

Use the _**Gradient panel**_ to create or edit gradients. Although the _**Gradient palette**_ allows you to select from pre-made gradients, you can also create new gradients to use or save.

(image: customizinggradients1-iphone.jpg width: 750)

Same settings can be found inside the (link: docs-iphone/effects text: _**Effects pane**_) from the Appearance panel whenever you select a _**Fill**_ with a gradient.

1. **Gradient Well** - Simple preview showing the current gradient. 

2. **Gradient Type** - Tap one of the three buttons to choose between _**Linear Gradient**_, _**Radial Gradient**_ or _**Angle Gradient**_. 

 (image: customizinggradients2-iphone.jpg width: 723)

3. **Angle** - Determines the rotation angle of the gradient.

4. **Reverse Gradient** - Reverse the order of the color stops in the gradient.

5. **Gradient Editor** - Create and edit new gradients using the gradient editor. Add gradient color stops by tapping a location on the gradient bar. Drag a color stop to change its location. Tap a gradient color stop to change its color. Drag a color stop away from the bar to easily remove it.
