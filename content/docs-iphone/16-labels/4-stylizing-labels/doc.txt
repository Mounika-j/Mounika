# Stylizing Labels

A labelâ€™s text font, size and style can be changed using the _**Style pane**_ inside the Label pane. Additionally, you can add and stylize a simple box bordering the label.

(image: stylizinglabels1-iphone.jpg width:750)

1. **Font** - Choose the font to be used for your label. Also, pick a style if the font allows it.

2. **Font Size** - Set the font size to be used for the label.

 (image: stylizinglabels2-iphone.jpg width:723)

3. **Text Color** - Choose the text color. 

4. **Background Color** - Set the background color. Set the color to **none** to make it invisible.

 (image: stylizinglabels3-iphone.jpg width:723)

5. **Margin** - The margin setting adjusts the size of the box bordering the text, adding space between the text and the edges.

6. **Corners** - Determines the roundness of the text box. Set this to **0 pt** if you wish to turn the background into a simple rectangle.

 (image: stylizinglabels4-iphone.jpg width:723)
